Ordered List

//Untuk Ordered List merupakan penempatan item pada html secara berurutan misal huruf ataupun angka. Dan untuk Ordered List merupakan kebalikan dari UnOrdered List yakni penempatan yang berurutan. Untuk membuat Ordered List dapat menggunakan perintah <ol> kemudian setelah itu <ol>

Codingan

<!DOCTYPE html> <html> <head> <meta charset="UTF-8"> <title> Order List </title> </head> <body style ="background:url(background.jpg); background-repeat: no-repeat; background-size: cover;"> <h1> <ol> <li> Masukkan 2 gelas belimbing air ke dalam panci </li> <li> Masukkan mie instan ke dalam panci </li> <li> Rebus mie instan selam kurang lebih 3 menit </li> <li> Masukkan bumbu dan minyak </li> <li> Siapkan mangkuk </li> <li> Tuangkan mie kedalam mangkuk</li> </ol> </h1> </body> </html>

Tampilan Hasil Percobaan

Kesimpulan

Pada percobaan Ordered List tag <ol> berfungsi untuk membuat list pada halaman web secara berurutan.
