Margin cell

//Tabel merupakan sebuah data dalam bentuk grid yang terdiri dari kolom, baris dan cell. Untuk membuat tabel kita perlu beberapa tag. Tag pertama yang kita gunakan adalah tag < table > dan diakhiri dengan tag < / table >, tag ini digunakan untuk memulai sebuah tabel. Tag kedua yang kita gunakan adalah tag < tr > dan diakhiri dengan < / tr > digunakan untuk membuat baris dari tabel. Tag ketiga yang kita gunakan adalahtag < td > dan diakhiri dengan < / td >digunakan untuk menginput data ke tabel. Sedangkan tag < th > digunakan untuk membuat judul sebuah table seperti nomer dan nama.Untuk format tabel bentuk tabel dapat menambahkan beberapa atribut seperti border untuk tebal garis tepi, align untuk posisi tabel secara horizontal, width untuk lebar tabel, dan height untuk tinggi tabel. Dan untuk margin cell sendiri digunakan untuk penggabungan beberapa sel menjadi satu, menggunakan perintah < td colspan="n"> kemudian keterangan yang akan dijadikan satu .

Codingan

<!DOCTYPE html> <html> <head> <meta charset="UTF-8"> <title> Menggabung Cell</title> </head> <body> <h1>Harga Buah </h1> <table border="2" align="center" width="40%"> <tr bgcolor ="#b2dfdb"> <td> Baris 1, Kolom 1 </td> <td> Baris 1, Kolom 2 </td> <td> Baris 1, Kolom 3 </td> </tr> <tr> <td> Baris 2 , Kolom 1</td> <td colspan="2"> Baris 2, Kolom 2 dan 3 digabung </td> </tr> <tr> <td> Baris 3, Kolom 1 </td> <td> Baris 3, Kolom 2 </td> <td> Baris 3, Kolom 3 </td> </tr> </table> </body> </html>

Tampilan Hasil Percobaan

Kesimpulan

Pada percobaan membuat margin ini dapat disimpulkan bahwa untuk tampilan hampir mirip dengan format tabel, hanya saja penambahan atribut coslan menjadikan nilai “2” pada colspan menunjukkan jumlah kolom ataupun baris yang akan digabungkan menjadi satu. Setelah kode HTML diatas dijalankan di browser, maka akan tampil seperti gambar di atas. Terlihat pada gambar diatas, pada baris 1 dan kolom 1 diterapkan colspan=”2″ yang menggabungkan dua kolom menjadi satu..
